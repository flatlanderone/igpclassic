# IZY Classic for 7 Days To Die 3.0.1 (version 3.0.6.0.1)

## Overview

**IZY Classic** is the official updated version of the IZY-All in One Gun Pack v5.1.3 modpack. Easier to install, to update and to maintain, it brings the classic weapon line-up to 7 Days To Die v3.0.

## What to install?

Before installing you need to download first the following files:

 1. IZY Classic - Core (see [Files section](https://www.nexusmods.com/7daystodie/mods/11059?tab=files)).
 2. IZY Classic - Resources (see [Files section](https://www.nexusmods.com/7daystodie/mods/11059?tab=files)).
 3. Flatlander's Combat Sandbox - available [here](https://www.nexusmods.com/7daystodie/mods/9980).
 4. Gears - available [here](https://www.nexusmods.com/7daystodie/mods/4017?tab=files).
 5. Quartz - available [here](https://www.nexusmods.com/7daystodie/mods/2409?tab=files).

**IZY Classic will not work without these required mods!**

## How to install IZY Classic manually?

Follow the instructions below to install IZY Classic manually.

**IMPORTANT: Do not add any of the files to Vortex, as it is not compatible with 7 Days To Die mods. We will not provide support or troubleshooting for Vortex installations.**

 1. Verify the integrity of 7 Days To Die game files. ([How to verify 7 Days To Die game files](https://www.nexusmods.com/7daystodie/articles/1623))
 2. Disable EAC ([How to disable EAC for 7 Days To Die](https://www.nexusmods.com/7daystodie/articles/1624))
 3. Download the files listed in the **What to install?** section above.
 4. Unzip them anywhere on your hard drive. 
 5. Find the 7 Days To Die **Mods** folder. By default the address is **%APPDATA%\7DaysToDie\Mods**. If necessary, create a **Mods**  folder in this location.
 6. Delete all the previous versions of the IZY-All in One Gun Pack from your **Mods** folder.
 7. Copy all the unzipped folders to the **Mods** folder.
 8. The Mods folder must contain the following folders:  
```

0-7D3.0-Izayo-WeaponFixes  
0-Alter_OverrideSounds  
0-Quartz  
00000-Gears  
IGP_Classic  
IGP_Resources  
zzzCombatSandboxFLTR

```
 
**IMPORTANT: Start a NEW GAME. We do not recommend adding IZY Classic to an existing save or removing it from an existing save. If you go against our advice, at least make a backup of your game save.**

## How to install IZY Classic automatically using Smörgåsbord?

IZY Classic and the required mods are available on [Smörgåsbord](https://www.nexusmods.com/7daystodie/mods/6764), a Wabbajack ModOrganizer2 ModList. Find out more about Smörgåsbord [here](https://www.nexusmods.com/7daystodie/mods/6764).

## Requirements

 - 7 Days to Die v3.0 and later. The mod is not compatible with 7 Days to Die v2.6.
 - EAC must be disabled.
 - All the required mods must be installed on the client and on the server.

## Issues and Bug Reports

**Before reporting an issue**:

 1. Make sure that you have followed the installation instructions on this page and that you installed ALL the required mods.
 2. If an issue appeared AFTER you added another mod to your game, remove that mod and see if the issue persists. Mod conflicts are NOT bugs. Talk to the author of that mod and let him know politely about the conflict.
 3. If an issue appeared AFTER you changed the vanilla Sandbox settings, tell us about these settings in your post.
 4. Always provide a log. See [How to find the log file and share it via pastebin.com](https://www.nexusmods.com/7daystodie/articles/1625).

## Feature Suggestions

## Important Notes

 - This page hosts the only official and authorized version of IZY Classic on Nexus. All full or partial re-uploads and modifications will be reported.
 - Do not spam Izayo or Flat Lander about release dates and bug patches. Harassment will be reported.
 - Before asking for help, ensure that you have followed the installation instructions on this page. Always provide a full log via a link to pastebin.com.
 - We only support STABLE versions of the current game version. If you missed/deleted old files, we will not resurrect archived files.
 - IZY Classic is NOT compatible with most overhaul mods. Instructions to convert Izy Classic to another overhaul will be provided upon request.
 - FNS is using Izy Classic with over 530+ others mods in Smörgåsbord. However, that doesn't mean it's compatible with everything. Other weapon, progression, and ammo mods may cause errors when combined with Izy Classic.
 - IZY Classic is not compatible with the following mods:
```
Escape From Tarkov Overhaul
Firearms Expansion 5
Brazmock's Archery
MoreShots
Pant's Weapons
```	
 - You may experience some mild to moderate visual issues with:
```
Custom Avatars (VRoid)
7DVR
LittleRedSonja & Mumphy armor mods.
```
